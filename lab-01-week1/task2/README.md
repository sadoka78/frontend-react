# Lab 01 – Week 1
## Lab 1.2: Настройка React-проекта и компонент Counter

### Описание
В данной лабораторной работе был создан React-проект и реализовано
минимальное приложение для демонстрации работы React как слоя представления.

Основной задачей является реализация компонента счётчика.

---

### Настройка проекта
- Проект создан с использованием Vite
- Используется React версии 18+
- Проект запускается локально через npm

---

### Компонент Counter
- Отображает числовое значение (начальное значение — 0)
- Содержит кнопки Increment и Decrement
- Использует хук useState
- Интерфейс обновляется автоматически при изменении состояния

---

### Как запустить
1. Установить зависимости:
   npm install
2. Запустить сервер разработки:
   npm run dev
3. Открыть ссылку, выданную в терминале

---

### Пояснение
React обновляет интерфейс на основе состояния компонента.
При изменении состояния происходит повторный рендер и обновление DOM.
