# Lab 01 – Week 1
## Lab 1.1: Декларативный и императивный подходы

### Описание
В данной лабораторной работе показана разница между императивным и декларативным подходами
к созданию пользовательского интерфейса.

Одна и та же функциональность реализована двумя способами:
с использованием jQuery и с использованием React.

Задача — реализовать подсветку текста при нажатии на кнопку.

---

### Задание 1: Императивный подход (jQuery)
- Создана простая HTML-страница
- Добавлена кнопка и абзац текста
- Подключена библиотека jQuery
- При нажатии на кнопку CSS-класс добавляется или удаляется вручную
- Происходит прямое управление DOM

---

### Задание 2: Декларативный подход (React)
- Создан React-проект
- Использован хук useState
- Состояние управляет отображением интерфейса
- Отсутствует прямое изменение DOM

---

### Как запустить

#### Версия на jQuery
1. Открыть HTML-файл в браузере
2. Нажать кнопку для включения и отключения подсветки

#### Версия на React
1. Установить зависимости:
   npm install
2. Запустить проект:
   npm run dev
3. Открыть ссылку локального сервера в браузере

---

### Вывод
Декларативный подход в React является более удобным и масштабируемым
по сравнению с императивным подходом на jQuery.
